# CursorFun
แอปพลิเคชันสำหรับแสดงเส้นวงกลมรอบเคอร์เซอร์เมาส์ เพื่อยกระดับการนำเสนองานให้ดูเป็นมืออาชีพและช่วยให้ผู้ฟังโฟกัสจุดสำคัญได้ดียิ่งขึ้น

Key Features:
- Fully Customizable: ปรับขนาด (Size), ความหนา (Thickness) และความโปร่งใส (Opacity) ของเส้นได้อิสระ
- Color Selector: กำหนดสีที่ต้องการได้ตามใจชอบ
- Lightweight: โปรแกรมขนาดเล็ก ทำงานลื่นไหล ไม่หนักเครื่อง

Security & Privacy / ความปลอดภัย
โปรเจคนี้พัฒนาขึ้นเพื่อทดลองระบบและแบ่งปันให้ใช้งานกัน:
- Safe & Clean: มั่นใจได้ 100% ว่าไม่มีโค้ดอันตรายหรือมัลแวร์แอบแฝง
- No Data Collection: ตัวแอปไม่มีการรับ-ส่งข้อมูลใดๆ ออกจากเครื่องของผู้ใช้ ใช้งานได้อย่างสบายใจในทุกสภาพแวดล้อม
